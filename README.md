This desciption is in a alpha state.

Hardware needed for this project:
1x Arduino Uno
1x funduino joystick shield v1 a
2x 8x8 WS2812b Matrix
cables to connect from the arduino uno to the first matrix and another set of cabvles to connect both matrixes to each other

The original code was developed by Mirko Pavleski: https://www.pcbway.com/project/shareproject/Colorful_Arduino_Tetris_Game_WS2812B_LED_Matrix_Tutorial_e1126df3.html

The original design for the funduino shield + arduino uno case was developed by knighttm: https://www.printables.com/model/1227937-arduino-uno-joystick-shield-case
